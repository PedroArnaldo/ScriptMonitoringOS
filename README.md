# Scrip Monitoring OS

Script to monitor operating system and server resource usage.

---

## 🚀 Start

### 📋 Prerequisites

#### Installing mpstat command in linux

- In Red Hat based Linux
```
sudo yum install sysstat
```

- In ubuntu based Linux
```
  sudo apt install sysstat
```

---

## 🔧 Installation

 ```
  git clone git@github.com:PedroArnaldo/ScriptMonitoringOS.git
 ```

---

## ⚙️ Executando

```
  bash monitoring.sh
```
